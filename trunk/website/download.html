From CVS

Check out the ProxyTunnel sourcecode from SourceForge's CVS server using the following command:

cvs -z3 -d:pserver:anonymous@cvs.proxytunnel.sourceforge.net:/cvsroot/proxytunnel co proxytunnel
or download the latest daily CVS snapshot here: http://cvs.sourceforge.net/cvstarballs/proxytunnel-cvsroot.tar.gz
